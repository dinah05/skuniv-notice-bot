# skuniv-notice-bot (서경대학교 공지사항 디스코드 알림 봇)
서경대학교 홈페이지의 공지사항을 자동으로 확인하고,
사용자가 지정한 '특정 키워드가 포함된' 공지가 등록되면 디스코드로 실시간 알림을 보내주는 자동화 봇입니다.

---

## 주요 기능

* 1. 서경대학교 전체 공지사항 자동 수집
* 2. 공지 제목 기반 키워드 필터링
* 3. 사용자가 지정한 키워드 포함 시 디스코드로 즉시 알림 전송
* 4. 중복 알림 방지

---

## 사용 기술

* **Python 3.10**
* **requests** – 웹 페이지 요청
* **BeautifulSoup4** – HTML 파싱
* **Discord Webhook** – 알림 전송
* **GitHub Actions** – 주기적 자동 실행

---

## 동작 구조

1. 서경대학교 공지사항 페이지(`https://www.skuniv.ac.kr/notice`)에 접속
2. 공지 목록 HTML을 파싱하여 제목과 링크 수집
3. 공지 제목에 설정한 키워드가 포함되어 있는지 검사하고,
4. 조건을 만족하면 디스코드 웹훅을 통해 알림 전송

---

## 설정 방법

### 1️⃣ 키워드 설정

```python
KEYWORD = "창업"
```

* 원하는 키워드로 자유롭게 변경 가능함!

---

### 2️⃣ 디스코드 웹훅 설정 (필수)

GitHub Repository → **Settings → Secrets → Actions** 에서
다음 환경 변수를 추가해야 함.

| Name              | Value       |
| ----------------- | ----------- |
| `DISCORD_WEBHOOK` | 디스코드 웹훅 URL |

---

## 실행 방법

### 로컬 실행 (학교 서버가 외부 서버 요청을 차단해서 로컬로 실행해야 돌아감ㅜ)

```bash
python main.py
```

### GitHub Actions 자동 실행

* 일정 주기로 공지를 확인하고 싶을 경우
  `GitHub Actions + cron` 설정을 통해 무인 자동 실행 가능
